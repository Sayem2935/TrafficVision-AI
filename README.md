# TrafficVision-AI

Smart Traffic Violation Detection System

Final Year Capstone + Research Project

## Overview

TrafficVision-AI is an AI-powered system for detecting motorcycle helmet violations from traffic videos.

The system uses YOLO-based object detection and a Flask web application to analyze traffic footage and generate violation evidence.

This project is developed as a Final Year Capstone Project and Research Project.

## Research Contribution

This project contributes:

• Custom traffic dataset collection  
• Helmet violation detection  
• Rider–motorcycle association  
• Evidence generation system  

The dataset will be used to train a custom helmet detection model.
